models.server_utils.setup_server(app: flask.Flask)

The assumption is that this script will be the ENTRYPOINT for the inference container. SageMaker will launch the container with the “serve” argument. We also have the option of using multiple models from this single container; only one model will be active at a time (i.e., this is not a Multi Model Server), so it can be selected by name using the “model” parameter.

Parameters:

app – The flask application to set up

Returns:

None

models.server_utils.detect_to_feature(fixed_object_bbox: List[float], fixed_object_mask: List[List[float]] | None = None, detection_score: float | None = 1.0, detection_type: str | None = 'sample_object') → Dict[str, str | list]

Converts the bbox object into a sample GeoJSON formatted detection. The model container does not normally provide the world coordinates, so they’re defaulted to 0,0 here since GeoJSON features require a geometry.

Parameters:

  • detection_type – Class assigned to the detection

  • detection_score – Confidence score assigned to the detection

  • fixed_object_mask – Polygon version of mask generated by detectron2

  • fixed_object_bbox – Bounding box to transform into a geojson feature

Returns:

dict: Dictionary representation of a geojson feature
